Center for Applied Internet Data Analysis Dataset

The 1as_caida_processed dataset represents the network of Autonomous Systems (AS) on the Internet, derived from CAIDA's (Center for Applied Internet Data Analysis) measurements. An Autonomous System is a collection of IP routing prefixes under the control of a single entity that presents a common routing policy to the internet. The dataset captures the connectivity between AS nodes as observed through BGP (Border Gateway Protocol) routing tables. The network includes:

  • 26475 Autonomous Systems (nodes), each corresponding to an independent network (e.g., ISPs, universities, large organizations).
  • 106,762 links (edges) representing peer-to-peer relationships or customer-provider relationships between two ASes.
  • The network is undirected after preprocessing: an AS connection is treated symmetrically, regardless of its original directionality in BGP paths.
