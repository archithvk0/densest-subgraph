As-733 Dataset

The AS733 dataset represents a snapshot of the Internet at the Autonomous Systems (AS) level, collected by the University of Oregon’s Route Views project in November 1997. Each node in the graph corresponds to an Autonomous System — a collection of IP networks managed by a single organization — and each undirected edge represents a direct BGP (Border Gateway Protocol) connection between two ASes. The network includes:

  • 6,474 Autonomous Systems (nodes).
  • 13,895 direct connections (edges) between ASes.
  • The network is undirected, where each connection is treated symmetrically.
